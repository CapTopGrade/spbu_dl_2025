### Описание тестов файла test_integrity.py

Можно запустить командой из терминала из корневой папки датасета:

```sh
python3 tests/test_integrity.py
```

1. *test_texts_filenames*

Проверяет, равно ли множество имён файлов из директории "texts" множеству значений колонки "filename" таблицы "metadata.tsv"

2. *test_beginnings_lines*

Проверяет, начинаются ли строки стихотворений (всех, кроме A573 и A652) со строчных символов

3. *test_trailing_spaces_lines*

Проверяет, содержатся ли на концах строк стихотворений пробелы в количестве от одного и более

4. *test_last_lines*

Проверяет, являются ли последние строки стихотворений пустыми или содержащими только пробелы в количестве одного и более

5. *test_cross_corpus_index*

Проверяет, равно ли множество значений колонки "uid" таблицы "metadata.tsv" подмножеству колонки "uid" таблицы "oeuvre.tsv"

6. *test_tags*

Проверяет, равно ли в стихотворении количество открывающих тегов количеству закрывающих тегов

7. *test_language_tag*

Проверяет, содержат ли имена тегов только латинские или только кириллические символы

8. *test_terminus_quem*

Проверяет, содержат ли колонки "terminus_ante_quem" и "terminus_post_quem" таблицы "metadata.tsv"

9. *test_npsp*

Проверяет, содержат ли тексты стихотворений "\xa0"

10. *test_redundant_spaces*

Проверяет, содержат ли текстовые колонки таблицы "metadata.tsv" избыточные пробелы

11. *test_leading_spaces*

Проверяет, содержат ли текстовые колонки таблицы "metadata.tsv" пробелы в начале значений

12. *test_trailing_spaces*

Проверяет, содержат ли текстовые колонки таблицы "metadata.tsv" пробелы в конце значений

13. *test_lifetime_publication*

Проверяет, содержит ли колонка "lifetime_publication" таблицы "metadata.tsv" значения, не равные "да" или "нет"

14. *test_link_uid*

Проверяет, содержит ли колонка "lifetime_publication" таблицы "metadata.tsv" значения, которых нет в колонке "link_uid" таблицы "metadata.tsv"

15. *test_volumes*

Проверяет, содержит ли колонка "volume" таблицы "metadata.tsv" значения, которых нет в колонке "volume" таблицы "metadata.tsv"

16. *test_n_lines*

Проверяет, содержит ли колонка "n_lines" таблицы "metadata.tsv" значения, не равные NA и ниже нуля

17. *test_n_words*

Проверяет, содержит ли колонка "n_words" таблицы "metadata.tsv" значения, не равные NA и ниже нуля
